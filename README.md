# Parcours Process / Opérations — Collectif Tariqa PRO

**Version de test.** Le 5e pilier de la méthode du Collectif Tariqa PRO : structurer sa boîte
une fois que l'offre vend.

Pour le fondateur qui n'a plus un problème de clients, mais un problème de charge — tout passe
par lui, rien n'est écrit, la boîte s'arrête s'il s'arrête.

## Prérequis

Le **parcours CTP** (persona → offre → positionnement → voix) :
https://github.com/ZakiTariqaPRO/ctp-plugin

## Installer

```
installe le plugin https://github.com/ZakiTariqaPRO/ctp-process-plugin
```

Puis : `lance le parcours process`

## Les 8 modules

Vision/mission/valeurs → plan stratégique → diagnostic du goulot → SOP → outils & données (CRM)
→ équipe → délégation & rythme → export.

Chacun produit un livrable **et** un outil d'implémentation. Coach Zaki accompagne et challenge
tout du long.
